# RecipeGenerator
This project is an AI-powered Recipe Generator that helps users create delicious and creative meals using the ingredients they already have. Users simply enter available ingredients, and the system generates personalized recipes using AI models like OpenAI. To ensure accuracy and offer alternative suggestions, the app also integrates with external services such as the Spoonacular API to fetch structured recipes, complete with instructions, images, and nutritional information. Built with a React frontend and a Node.js backend, the app supports real-time recipe streaming, clean UI interactions, and secure API key management. The goal of this project is to reduce food waste, inspire home cooking, and make recipe creation smarter and more accessible.
